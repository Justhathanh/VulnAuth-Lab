VULN: Missing Secure + HttpOnly + SameSite

Insecure Cookie Demo

Cookie auth_token vừa được set không có security flags.

Mở browser console gõ document.cookie — đọc được token ngay.

Vulnerable code
resp.set_cookie("auth_token", "user_session_abc123",
                httponly=False,
                secure=False,
                samesite=None)

# Fix:
resp.set_cookie("auth_token", token,
                httponly=True,
                secure=True,
                samesite="Strict",
                max_age=3600)
